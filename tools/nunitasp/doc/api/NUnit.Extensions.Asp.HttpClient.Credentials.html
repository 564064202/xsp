HttpClient.Credentials Field

Username and password (null if none). Set automatically if the username and password are supplied in the URL (i.e., "http://username:password@host"). Can be used with both "basic" and "Windows Integrated" (NTLM) authentication methods. Set this property to 

CredentialCache.DefaultCredentials
to use your current Windows login.
